# REAL-TIME-THEFT-DETECTION
I’m working on a project to detect real-time theft using Yolov10 models! I’ve trained four custom models: Yolov10-n, Yolov10-s, Yolov10-m, and Yolov10-l.
Each one has its own set of parameters, so they can easily be linked with CCTV cameras or other cameras. This way, you can use them in real-time with minimal delay, or you can also use them on recorded footage.
